# Samaj Documentation

Samaj is a safety-first Python GUI platform for lawful, authorized security
learning, defensive assessment planning, evidence management, and reporting
workflows.

Current documentation covers Phase 0, Phase 1, and the Phase 2 foundation:

- Installation with mandatory project-local `.venv`
- Windows and Linux/Kali start/stop scripts
- Safety requirements
- Anti-Hallucination requirements
- Planned module boundaries
- Neutral Tool Catalog / Knowledge Registry intake
- Tool Catalog disclaimer acceptance
- OS support mapping fields
- Static-only Safe Analysis Lab foundation
- Proxy database foundation
- Modern Samaj theme and logo foundation
- Plugin development boundaries
- Reporting boundaries

Active testing, external tool execution, recon, OSINT, report export, plugin
execution, AI provider integration, dynamic malware sandboxing, live proxy
testing, and proxy-chain execution are planned but not implemented yet.
