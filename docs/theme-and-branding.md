# Theme and Branding

The current GUI includes a modern Samaj theme foundation and a local logo asset.

## Assets

- `samaj/gui/assets/logo.png`
- `samaj/gui/theme.py`
- `samaj/gui/widgets/navigation_sidebar.py`
- `samaj/gui/widgets/modern_card.py`

The logo is used in the sidebar and About page. The theme keeps the safety
banner highly visible and keeps operational pages quiet, compact, and readable.

## Boundary

Theme work does not change the safety model. Safe Mode, Anti-Hallucination
Mode, scope requirements, disclaimer acceptance, and audit logging remain active.
