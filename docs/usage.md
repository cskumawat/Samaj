# Usage

Launch the GUI:

```bat
start.bat
```

Stop the running GUI process started from this project:

```bat
stop.bat
```

The GUI currently provides:

- Dashboard
- Settings
- Logs and audit trail
- Projects
- Tool Catalog / Knowledge Registry with disclaimer acceptance
- Safe Analysis Lab static-only page
- Proxy Database page
- About Samaj page
- Placeholder navigation for future modules
- Permanent Safe Mode and Anti-Hallucination banner

The GUI does not run catalog tools. It does not run active recon, scans,
dynamic malware analysis, live proxy testing, proxy chains, plugins, or AI
provider calls in this phase.

The Project Manager can create, edit, list, and archive local project records in
the Phase 2 SQLite database.
