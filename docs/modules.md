# Modules

## Implemented Foundation

- GUI shell
- Settings
- Audit log view
- Safety policy primitives
- Anti-Hallucination primitives
- SQLite database initialization
- Additive SQLite schema repair for current foundation columns
- Project CRUD service and basic GUI
- Scope, asset, finding, evidence, and audit CRUD services
- Phase-wise Tool Catalog / Knowledge Registry intake
- Tool Catalog disclaimer acceptance gate
- OS support mapping fields for catalog entries
- Safe Analysis Lab static-only analyzer foundation
- Proxy database parsing/import/storage foundation
- Modern Samaj theme and logo foundation

## Not Implemented Yet

- Active recon
- Active OSINT collection workflows
- DNS probing
- HTTP probing
- Screenshot capture
- Fingerprinting execution
- Port scan orchestration
- Manual testing workbenches that run tools
- Full evidence manager GUI
- Full findings GUI
- Report export automation
- Executable tool registry
- Plugin manager execution
- Freeze manager automation
- Dynamic malware sandboxing
- Live proxy testing
- Proxy-chain execution

Planned modules must not claim support until their implementation and tests
exist.

## Current Boundary

Catalog tools are knowledge entries only. A catalog entry does not mean Samaj
supports, endorses, verifies, recommends, installs, executes, or grants
permission to use a tool.
